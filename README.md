## vann.io

This is a static blog site built using Jekyll and Gulp.

### Dependencies

- [Node.js](//nodejs.org/)
- [Gulp](//gulpjs.com/)
- [Sass (3.4+)](//sass-lang.com/install)
- [Ruby (2.0+)](//ruby-lang.org)
- [Jekyll](//jekyllrb.com/)

### Installation

1. Install dependencies listed above
2. `bundle && npm install`
3. `gulp`
