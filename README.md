## vann.io

This is a static blog site built using Jekyll and Gulp.

### Dependencies

- [Node.js](http://nodejs.org/)
- [Gulp](http://gulpjs.com/)
- [Sass (3.4+)](http://sass-lang.com/install)
- [Jekyll](https://jekyllrb.com/)

### Installation

1. Install dependencies listed above
2. `npm install`
3. `gulp`
